# Melhores Práticas - Coleção Pokémon

Este diretório contém as diretrizes e melhores práticas para o desenvolvimento do site de coleção de cartas Pokémon.

## Arquivos de Documentação

- `codigo-html.md` - Padrões de código HTML
- `bootstrap-guidelines.md` - Uso correto do Bootstrap
- `estrutura-paginas.md` - Estrutura de páginas individuais de cartas
- `acessibilidade.md` - Práticas de acessibilidade

## Princípios Gerais

1. **Zero CSS Customizado**: Usar apenas classes Bootstrap 5
2. **Semântica HTML**: Usar tags HTML5 apropriadas
3. **Responsividade**: Mobile-first design
4. **Consistência**: Manter padrões em todas as páginas
5. **Acessibilidade**: Seguir WCAG 2.1 AA
